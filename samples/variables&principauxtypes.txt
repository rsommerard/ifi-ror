Introduction :
Ce fichier a pour but de vous apprendre comment utiliser les variables en Ruby ainsi que de connaître les trois principaux types existant.

Exercice 1 - Déclaration et typage des variables
Exécuter ces deux lignes:
a = 10
a = "Bonjour"

Exercice 2 - Trois principaux types
a = 10
b = 10.5
c = "Coucou"
Quels sont les types de chaque variable ?
Vous pouvez saisir le nom de la variable, et rajouter .class au bout.
Exemple: a.class
